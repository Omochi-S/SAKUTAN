# SAKUTAN
英単語暗記アプリ「サクタン」

「SAKUTAN.apk」は、アンドロイド機種にインストールできるデータです。
「SAKUTAN.aia」は、MIT app Invetor 2でのプログラムデータです。

「サクタン」は、第一回学力向上コンテストに応募したandroidアプリの作品です。
「第一回学力向上アプリコンテスト」にて、アイディア部門優秀賞を受賞しました。
 
　「サクタン」は楽しく効率的に英単語を暗記することを目的に、フラッシュカード式の単語帳に加えて、英単語を楽しく暗記できるもぐらたたきゲーム機能を搭載しています。また、ゲームは目覚まし時計と連動させることができるため、記憶のアウトプットに有効な朝の時間帯を有効に活用することができます。
　制作にあたって暗記学習に関してのアンケート調査・分析を行い、継続的に楽しく使用してもらえるように工夫しました。
